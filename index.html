How to publish

  1. Fork the registry repo.
  2. Place your downloaded .json file in the domains/ folder.
  3. Submit a Pull Request.
  4. Configure your host (GitHub/Vercel) to accept the new domain.
  5. If you choose the REDIRECT type, you do not need to do any setup on your website, only submit the PR here.
