Feature
Discord Community Redirect
Enter a Thunderstore community ID to be redirected to their Discord server. This works via URL too, try linking to modding-community.com/#community-id to redirect visitors straight to a community's Discord. Credits for this feature go to https://scyye.dev/

How it works

  1. Uses the Thunderstore ecosystem schema to find community Discord links.
  2. Linking to modding-community.com/#peak auto-redirects to that community's Discord.
  3. Works with any valid Thunderstore community ID.
